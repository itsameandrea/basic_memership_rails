Educational Project from Udemy.com

The application matches developers with enterpreneurs in order for them to make business. The whole backend has been developed using RoR whilst the front end is using HTML/CSS/Javascript(JQuery).

The app includes two membership types:

- Basic
- Pro (10 euro/month subscription)

The payment has been handled with Stripe.

The app has been deployed on Heroku and is currently live here -> https://shrouded-taiga-63589.herokuapp.com/
